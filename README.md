# Mirza Portfolio (Next.js + shadcn/ui + Bun)

A clean, modern developer portfolio built with Next.js (App Router), shadcn/ui, Tailwind CSS, and Bun.

## Getting started

From the repo root:

```bash
cd portfolio
bun install
bun run dev
```

Then open `http://localhost:3000`.

## Customize your content

Edit your details, links, projects, skills, and experience in:

- `lib/site-config.ts`

The page is implemented in:

- `app/page.tsx`

## Production build

```bash
cd portfolio
bun run build
bun run start
```
